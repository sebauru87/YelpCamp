Enviroments Variables
el entorno donde se corre la pagina y la base de datos, ej en heroku o localhost
asi tenemos 2 bases de datos diferentes
tambien sirve para que la gente no sepa el codigo de nuestra base de datos, sea invisible, que solo se vea DATABASEURL

//in the command line
export DATABASEURL=mongodb://localhost:27017/yelp_campV8

and then in your app.js
mongoose.connect(process.env.DATABASEURL);

//este para el database local
